# huhu23
Model to participate un HUHU-2023 Iberlef.

## Run and Train

First install dependencies
```shell
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

to train and predict with our best model

```shell
python main.py --modeltype ted --balance
```

for more help use
```shell
python main.py --help
```
